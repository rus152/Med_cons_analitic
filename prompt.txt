Так же тебе надо провести подытог, 

Вовлечённость врача	(Слабая, средняя, сильная)
Вовлечённость пациента	(Слабая, средняя, сильная)
Выполненость чек-листа	(число выполненых пунктов)/(число пунктов)

Проанализируй каждый ответ на его соответствие, что бы не ошибиться в своём ответе. Необхадим качественный анализ для дальнейшего анализа, насколько хорошо врач справился со своими обязанностями, если ты неправильно оценишь ситуацию, этот врач может получить по голове за то, что он якобы что-то не спросил. По этому внимательно, как будто бы ты сам этот врач, проверь и удостоверься в пунктах.

Ответ ты должен выдать строго в формате json:

[
	{
      Questions:[
        {
            id: '(Номер вопроса)',
             Results:[
               { 
                 Question_text: '', 
                 result: '' 
               }
             ]
      	}      
     	] 
      Doctors_involvement:'(Слабая, средняя, сильная)',
      Patient_engagement:'(Слабая, средняя, сильная)',
      Completion_of_the_checklist:'XX/YY ZZZ%';
	}
]

XX = Сколько выполненых условий
YY = Количество условий(Так же из этого числа вычти 5, так как 5 вопросов не относятся к врачу)
ZZZ = На сколько процентов выполнено
